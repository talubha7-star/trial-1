## Quick Start

1. Install Node.js (v18+).
2. Install dependencies:
   ```bash
   npm install
   # or: pnpm install / yarn
   ```
3. Run locally:
   ```bash
   npm run dev
   ```
4. Deploy:
   - **Vercel**: Import this repo in Vercel and deploy (auto build).
   - **Netlify**: New site from Git, build command `npm run build`, publish directory `.next` (use Netlify Next.js adapter) or use the Netlify Next.js runtime.

## Customizing
- Search for "YourBrand" and replace it with your brand name.
- Update contact form action with your Formspree endpoint or swap to any provider.
- Edit content in `app/page.tsx` and global styles in `app/globals.css`.
